# Java Collections Practice

This repository contains three small Java projects that I created to practice and revise the Collection Interface concepts.

---

## Concepts Covered

- Collection Interface hierarchy
- List (ArrayList)
- Set (HashSet)
- Map (HashMap)
- Adding and removing elements
- Iterating using for-each loop and Iterator
- Sorting using Comparator
- Handling duplicates
- Working with key-value pairs

---

## Project 1: Student List Program

This project uses **ArrayList** to store Student objects.

Features:
- Add students
- Update student marks
- Remove students
- Display students using Iterator
- Sort students by name

---

## Project 2: Unique Number Checker

This project uses **HashSet** to store numbers.

Features:
- Accept numbers from user
- Automatically remove duplicate values
- Display unique numbers
- Show total unique count

---

## Project 3: Simple Dictionary

This project uses **HashMap** to store words and their meanings.

Features:
- Store key-value pairs
- Search meaning using a word
- Check if a key exists
- Iterate using entrySet()
